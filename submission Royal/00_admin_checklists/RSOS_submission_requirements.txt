Royal Society Open Science submission requirements

Source
Royal Society Open Science author guidance:
https://royalsocietypublishing.org/rsos/for-authors

1. Manuscript file
- Initial submission may be a Word or PDF file with embedded figures.
- Manuscript language must be English.

2. Core manuscript metadata
- Title
- Author names
- Affiliations
- Corresponding author details
- Abstract
- Keywords
- Article type

3. Required declarations
- Funding statement
- Competing interests statement
- Author contributions statement
- Ethics statement (if applicable)
- Data accessibility statement

4. Data and code availability
- Supporting data/code/materials should be publicly available.
- Repository access should not be "available on request" only.
- DOI-backed repository deposition may be required by revision stage.

5. Figures and tables
- Embedded figures are acceptable for initial review.
- Separate figure/table files may be required at production.

6. Suggested referees
- Provide at least 4 suggested referees.
- Referees should be independent and active in the field.
- Opposed referees can be listed with reasons.

7. Submission policies
- Submit via the journal online submission system.
- Confirm open access license terms.
- Transparent peer review is default unless opt-out is approved.

8. Revision stage
- Prepare a response-to-reviewers document and tracked-changes manuscript.

Project checklist
- Complete manuscript placeholders (authors, affiliations, corresponding author, registry ID, funding, conflicts, contributions)
- Finalize repository link/DOI in data accessibility statement
- Add any required data/code references
- Prepare suggested referee list
